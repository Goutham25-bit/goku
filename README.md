# Goutham's Portfolio Website

A dark & bold personal portfolio website with a Node.js backend and MongoDB database.

## Live Site
https://goutham-portfolio.onrender.com

## Tech Stack
- HTML, CSS, JavaScript (Frontend)
- Node.js + Express (Backend)
- MongoDB Atlas (Database)
- Deployed on Render

## How to Run Locally
1. Run `npm install`
2. Add your MongoDB connection string to `server.js`
3. Run `npm start`
4. Open `http://localhost:3000`
